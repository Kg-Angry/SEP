# Sistem elektronskog placanja

**Uvod:**

Predmetni projekat podrazumevao je razvoj Koncentratora placanja - web aplikacije koja simulira placanja na internetu. Pored Koncentratora placanja neophodno je bilo implementirati vise nacina placanja kao sto su Pay-Pal, Bitcoin i Banka (placanje debitnom karticom). Klijent za pomenutu web aplikaciju bice Naucna centrala koja se preuzima sa predmeta Upp i Udd.

**Radno okruzenje:**

* Intelij IDEA

**Tehnologije:**

* Spring Boot 2.2.0 Release
